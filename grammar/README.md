# Grammar

## Definitions

Definitions are basic EBNF groupings that describe a collection of characters like numbers or letters.

## Literals

Literals represent a pure value of a specific type that is known at compile time.

- Boolean literal
- Symbol literal
- String literal
- Number literal
  - Binary notation
  - Octal notation
  - Decimal notation
  - Hexadecimal notation
  - Scientific notation

## Comments

Comments can be either single line or span multiple lines and are not evaluated during compilation (only in the lexical
analysis).
